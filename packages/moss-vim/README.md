# moss-vim
Vim/Neovim syntax highlighting for [Moss](https://github.com/moss-lang/moss).

## Installation
With [vim-plug](https://github.com/junegunn/vim-plug):
```vim
Plug 'moss-lang/moss', { 'rtp': 'packages/moss-vim' }
```

## Disclaimer
This plugin was ported from the [VS Code extension](../moss-vscode) using Claude. If the grammar gets out of date, you can ask Claude to update it — see `CLAUDE.md` for context.
