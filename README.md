# Quench

A programming language.

Prerequisites:

- [Rust][]
- [Tree-sitter CLI][]

First install:

```sh
cargo install --locked --path .
```

Then run an example:

```sh
examples/hello.qn
```

## Editors

Support (and instructions) for specific text editors can be found in the
[editors][] folder.
