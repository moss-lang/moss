quench 0.3.0

Here is an example Quench program:

    #!/usr/bin/env quench
    print("Hello, world!");

Save the above contents as hello.qn and run this command:

    quench run hello.qn

Or on a Unix-like system, you could instead run these two commands:

    chmod +x hello.qn
    ./hello.qn

Either way, you should see this output:

    Hello, world!

USAGE:
    quench [FLAGS] <SUBCOMMAND>

FLAGS:
    -h, --help                  Prints help information
        --stdlib-placeholder    Replaces stdlib import with an example.com placeholder
    -V, --version               Prints version information

SUBCOMMANDS:
    compile    Compiles a file to JavaScript
    help       Prints this message or the help of the given subcommand(s)
    lsp        Starts the language server
    run        Runs a script
